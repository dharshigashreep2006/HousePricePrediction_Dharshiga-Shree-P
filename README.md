# House Price Prediction

This project focuses on predicting house prices using machine learning techniques. The dataset was cleaned and analyzed, and two models were implemented using Scikit-learn:

* Linear Regression
* Random Forest Regressor

### Key Findings

* Area, bathrooms, parking spaces, and air conditioning have a strong influence on house prices.
* Linear Regression achieved better performance with an R² score of approximately 0.65.
